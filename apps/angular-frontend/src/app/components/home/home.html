Bienvenue dans notre application Angular + NestJS avec gRPC

Cette application démontre la communication entre un frontend Angular et un backend NestJS en utilisant gRPC pour la communication inter-services.

Fonctionnalités

🚀 Angular SSR

Application Angular avec Server-Side Rendering pour de meilleures performances et SEO

⚡ Signaux Angular

Utilisation des signaux Angular pour une gestion d'état moderne et réactive

🔌 gRPC

Communication haute performance entre les microservices avec gRPC

🏗️ NestJS

Backend robuste et scalable avec NestJS et support des microservices
